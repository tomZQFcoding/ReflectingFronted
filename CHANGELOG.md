## 📝 使用指南

### 暗色模式
- 设置 → 主题切换按钮
- 自动检测系统偏好

### 高级搜索
- 点击搜索框旁的搜索图标
- 设置多个筛选条件
- 点击"搜索"应用筛选

### 数据导出
- 设置 → 导出备份 (JSON)
- 设置 → 导出 Markdown
- 复盘详情 → 导出为图片

### 目标追踪
- Dashboard → 目标追踪卡片
- 创建、编辑、完成、删除目标

### 月度报告
- Dashboard → 月度报告卡片
- 使用左右箭头切换月份
